GITHUB

A series of small, low overhead games I built (and am building) to demonstrate to power of Javascript and the HTML5 canvas to build simple prototypes of games. Each of these games has a focus on simple implementation, so many of them use generated shapes for graphics over image sprites. In addition, they are not optimized for speed or efficiency - rather the goal was to build simple games quickly. Click an image to play that game
